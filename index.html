HTML TAGS

S.NO Tag Description Example
1 Header Headliner with font size 1 to 6 h1
2 Bold Makes the font bold b
3 Italic Makes the font italic italic
4 Underline Underlines the text underline
5 Ordered list Creates a list with a numerical order of 123 or roman numerals ol
6 Unordered list Creates a unordered list with shapes like circle, square, etc ul
7 Anchor Creates a link to link a webpage in the html or link to a subpage created by the user a
8 Age tag Embed a image copied from web or ember a local image img
9 Division Creates a specific division to segregate the contents div
10 Table Creates a table to include the header and body contents table
11 Caption Creates a specific caption for a given table caption
12 Button Breated a button for the user to submit a form button
13 Figure caption Creates a caption that can be used for a figure figcaption
14 Form Used to create a form that can be used by the owner to get user data form
15 Break Breaks the line and send the next information to the next line break
16 Navigation Used to represent a section with navigation link or group a section with links nav
17 Input Most used tag in form to define the type of input used in form which varies from radiobutton,checkbox,etc input
18 Label Label defines a label for many form elements and is useful for the users label
